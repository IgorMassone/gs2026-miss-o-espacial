##Sobre o Projeto

Sistema inteligente de monitoramento para controle de missões espaciais experimentais, desenvolvido como solução para o desafio **Global Solution 2026 — Soluções em Energias Renováveis e Sustentáveis** da FIAP.

O sistema recebe, interpreta e exibe dados simulados das condições operacionais de uma missão espacial, incluindo temperatura, energia, comunicação e status dos módulos. Diante de situações críticas, o sistema gera alertas automáticos e executa respostas automatizadas sem intervenção manual.

---

##Funcionalidades

- Monitoramento de temperatura, energia, comunicação, módulos e fonte energética (solar, bateria, eólica, nuclear)
- Geração automática de alertas para condições críticas
- Respostas automatizadas: resfriamento emergencial, modo econômico, reconexão automática, isolamento de módulo
- Histórico de leituras organizado com linked list (arrays paralelos)
- Análise operacional: médias, tendência de temperatura, risco da missão
- Avaliação de sustentabilidade energética: prioridade de fontes renováveis

##Como usar

1. Faça o download do arquivo `missao.html`
2. Abra no navegador
3. Preencha os campos com os dados da leitura e clique em **Adicionar**
4. Use os botões **Status**, **Histórico** e **Análise** para monitorar a missão
5. Clique em **Encerrar** para finalizar a sessão
